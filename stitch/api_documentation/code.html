Single Prediction

Analyze a single text string for potential cyberbullying content. Our NLP engine classifies the text into various categories including hate speech, aggression, and personal attacks.

POST https://api.cyberguard.io/v1/predict

Request Body json 

{
  "text": "You are so stupid, why do you even exist?",
  "context_id": "thread_92834",
  "model_version": "latest",
  "include_scores": true
}

Response Example 200 OK 

{
  "status": "success",
  "request_id": "req_fb2930485",
  "prediction": {
    "is_cyberbullying": true,
    "confidence": 0.984,
    "categories": [
      { "label": "insult", "score": 0.92 },
      { "label": "personal_attack", "score": 0.87 }
    ],
    "flagged_tokens": ["stupid", "why do you even exist"]
  },
  "processing_time_ms": 142
}

Properties

Field Type Description
text string The content to be analyzed. Max 5000 characters.
context_id string Optional. Unique ID for tracking user sessions or threads.
include_scores boolean Return confidence scores for each sub-category. Defaults to false.
